Présentation du projet

Dans le cadre de mon BAC PRO CIEL (Cybersécurité, Informatique et réseaux, ÉLectronique), j'ai développé un jeu de morpion jouable en ligne contre un bot intelligent.

Le bot utilise une logique heuristique : il sait gagner quand l'occasion se présente, bloquer le joueur, et occuper les meilleures cases stratégiques. Il n'est pas imbattable — un joueur attentif peut le piéger avec une double menace.

Le site a été entièrement codé en HTML5, CSS3 et JavaScript pur, sans aucun framework ni bibliothèque externe.
